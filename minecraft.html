Server whitelist is enabled. Contact a moderator if you want access to be granted to yourself or your friends.

Server rules:

  • All (large) automated farms must have an on/off switch and should be turned off when you leave the server.
  • Similarly, try to minimize entity counts in farms and bases. Do not leave your boats behind while exploring.
  • Server economy and currency is heavily discouraged. Instead, consider trading resources or favours!
  • While pranking is allowed, do not irreversibly change or damage someone else's builds without their explicit permission. Do not steal or copy items from other players without permission and make sure to replace any items you borrow.
  • Do not mod, hack, cheat or exploit bugs to gain an advantage that would not normally be available in regular gameplay. (Visual mods such as MiniHUD or Litematica are allowed!)
  • Racism, sexism or discrimination on any other basis will not be tolerated.
  • Everyone must comply by the usage guidelines as set by Mojang Studios and Microsoft.
  • Server mods hold the right to deny or revoke access to the server for any player at any time.

To streamline multiplayer gameplay and increase server performance, several changes have been made to the vanilla Minecraft mechanics. Gameplay changes are as follows:

  • The following gamerules have changed:
    • playersSleepingPercentage has been set to 51%.
    • ExplosionDropDecay has been set to FALSE for all sources.
    • simulation-distance has been reduced to 8.
  • Mobs can be silenced using name tags. 'Silence me', 'silence me' and 'silence_me' are all acceptable names.
  • Players can type '/trigger spawn' to teleport to spawn.
  • Players can type '/trigger ch_toggle' to show a small HUD showing current coordinates, direction and in-game time of day.
  • To assist in the synchronization of nether portals, players can type '/trigger nc_inOverworld' to convert their current coordinates to nether coordinates, or '/trigger nc_inNether' to convert them to overworld coordinates.
  • Shulker shells have been made more accessible by increasing their drop rate to two per shulker.
  • Elytra and dragon eggs are now renewable through slaying of the ender dragon.
  • Crafting recipes have been added for name tags, horse armour, leather (smelting), blackstone, gravel, and enchanted golden apples. Additionally, all unpackables have been added, as well as all dye related recipes and 'back to blocks'.
  • Finally, concrete can be crafted in bulk by dropping stacks of concrete powder into a cauldron filled with water.

Currently, voice chat and proximity voice chat are not implemented on the server, as this would require players to install mods to use certain server functionality. Though this topic is still open for discussion, for now you can make use of the discord voice-call functionality or connect through other third-party means.

For the purpose of finding biome types or slime chunks, the server seed is -4935325944529060349. Please keep things fun for yourself and limit the search for structures in this way.
